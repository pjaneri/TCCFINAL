# **App Name**: Recycle+ (Recicla Mais)

## Core Features:

- Login: Permitir que os usuários acessem o aplicativo com nome de usuário e senha.
- Criar Conta: Permitir que novos usuários criem uma conta no aplicativo.
- Redefinir Senha: Permitir que os usuários redefinam sua senha caso a tenham esquecido.
- Registro de Reciclagem: Permitir que os usuários registrem os produtos que reciclaram.
- Visualização de Pontos: Mostrar aos usuários seus pontos acumulados com base na reciclagem.
- Resgate de Prêmios: Permitir que os usuários resgatem seus pontos por prêmios.

## Style Guidelines:

- Cor primária: Vermelho escuro (#8B0000) para representar a identidade visual do SENAI e inspirar ação.
- Cor de fundo: Branco acinzentado suave (#F0F0F0) para proporcionar uma leitura confortável e clean.
- Cor de destaque: Um tom de laranja vibrante (#FFA500) para botões de ação e notificações importantes.
- Fonte para títulos: 'Poppins', sans-serif, para um visual moderno e legível.
- Fonte para corpo do texto: 'PT Sans', sans-serif, para garantir a clareza na leitura dos textos e informações.
- Ícones claros e intuitivos que representem as diferentes ações e produtos reciclados.
- Layout simples e organizado, priorizando a facilidade de uso e a acessibilidade das informações.
